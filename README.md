# Taller de Internet I

Este es el repositorio de la materia **Taller de Internet I** de la carrera **Licenciatura en Ciencia de Datos**.

Su objetivo es servir como espacio de trabajo para las **prácticas de la materia**, incluyendo ejercicios, actividades, trabajos grupales y el desarrollo del proyecto integrador.

## Información general

- **Carácter:** Obligatoria
- **Dedicación:** Cuatrimestral
- **Modalidad:** Presencial
- **Carga horaria:** 48 hs
- **Horas teóricas:** 25 hs
- **Horas prácticas:** 23 hs
- **Docentes:** Carmona, Augusto / Saucedo, Leonardo

## Descripción de la asignatura

La asignatura introduce a los estudiantes en los fundamentos técnicos y conceptuales del desarrollo web, entendiendo a Internet como una plataforma clave para la generación y explotación de datos.

Durante la cursada se abordan:

- Principios de funcionamiento de la web.
- Lenguajes de marcado y estilo (HTML y CSS).
- Programación con JavaScript.
- Manipulación del DOM.
- Consumo de APIs y trabajo con datos reales.
- Uso de herramientas colaborativas de control de versiones (Git y GitHub).

Como instancia de cierre, se desarrolla y despliega un proyecto integrador que consolida los contenidos trabajados.

## Objetivos

- Entender que la web es una fuente con gran cantidad y variedad de datos.
- Comprender la estructura y el funcionamiento de la web (protocolos, lenguajes y arquitectura de información).
- Dominar técnicas para construir conjuntos de datos estructurados a partir de información web.
- Aplicar conocimientos en proyectos de scraping.
- Trabajar colaborativamente utilizando herramientas de versionado.

## Unidades temáticas

1. **Introducción al desarrollo web y versionado de código con HTML**
   - Funcionamiento web básico (cliente-servidor, navegador, URL, HTTP).
   - Estructura de documentos HTML, etiquetas y semántica.
   - Introducción a Git y GitHub.

2. **CSS y diseño web**
   - Selectores, modelo de caja, tipografía y colores.
   - Flexbox, Grid, media queries y diseño responsive.

3. **Fundamentos de programación con JavaScript**
   - Tipos de datos, variables, operadores.
   - Condicionales, ciclos y funciones.

4. **DOM, APIs y datos**
   - Selección y modificación de elementos del DOM.
   - Eventos e interactividad.
   - Solicitudes HTTP, JSON e integración de datos externos.

5. **Proyecto integrador**
   - Desarrollo y despliegue de un portal de noticias web con perfiles de usuario y administrador, consumiendo información en tiempo real mediante APIs.

## Metodología

Las clases son teórico-prácticas:

- Exposición de contenidos por parte del equipo docente.
- Resolución de ejercicios de aplicación en clase.
- Trabajo grupal y presentaciones.
